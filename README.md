# rannoy

[spotify/annoy](https://github.com/spotify/annoy) bindings for Rust.

```toml
[dependencies]
rannoy = { git = "https://github.com/uzushino/rs-annoy" }
```

## Building

```sh
$ git clone https://github.com/uzushino/rannoy
$ cd rannoy
$ cargo build
```

## Example

### mnist

```
$ cargo run --example mnist
```
